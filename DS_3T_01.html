Comme le lièvre se voie perdre, parfois, contre la tortue il lui a proposé de refaire la course en changeant un peu les régles. On vous donne ci-après l'algorithme de la course.

Algorithme Lievre_tortue_v2
Début
  // Mettre les joueurs en position de départ
  pos_lievre ← 0
  pos_tortue ← 0
  Répéter
    // Lancer le dé
    de ← alea(1, 6)
    Ecrire("dé =", de)
    // Avancer les joueurs en fonction du dé
    Si de ≠ 6 Alors 
      pas ← (de + 1) div 2
      pos_tortue ← pos_tortue + pas
      Si pos_tortue > 8 Alors
        pos_tortue ← 8
      Fin si
      Ecrire("Position tortue :", pos_tortue)
    Sinon
      pos_lievre ← pos_lievre + 4
      Ecrire("Position lièvre", pos_lievre)
    Fin Si
  // Répéter les opérations précédentees
  // Jusqu'à l'un des joueurs atteigne la position d'arrivée
  Jusqu'à (pos_lievre = 8) ou (pos_tortue = 8)
  // Afficher le gagnant
  Si pos_tortue = 8 Alors
    Ecrire("La tortue gagne.")
  Sinon
    Ecrire("Le lièvre gagne.")
  Fin Si
Fin

Exercice 1 (10 pts)

Bandit Manchot

C'est un jeu simple avec trois rouleaux de neuf symboles comportant, souvent, neuf images. (On suppose que les images sont numérotées de 1 à 9)

La machine fonctionne comme suit :

  • Le joueur mise un montant en dinars qu'il introduit dans la machine
  • Il tire un levier, les rouleaux se mettent à tourner puis s'arrêtent au bout d'un temps.

Les combinaisons gagnantes sont :

111

gain = montant * 10

11c, 1b1, a11

gain = montant * 5

55c, a55, 5b5

gain = montant * 2

Autres

gain = 0

Remarque :

  • a, b et c désignent des chiffres quelconques.
Travail demandé

Ecrire un algorithme nommé bandit_manchot qui permet de :

  • Saisir le montant misé
  • Générer trois valeurs aléatoires a, b et c entre 1 et 9
  • Calculer le nombre abc à afficher par la machine
  • Calculer le gain selon les valeurs a, b et c d'après le tableau précédent
  • Afficher le gain du joueur
